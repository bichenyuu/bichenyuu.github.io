# Chenyu Bi

Source code for [bichenyuu.github.io](https://bichenyuu.github.io), Chenyu Bi's personal academic website.

The site is built with Jekyll and published by GitHub Pages.

## Site content

- `_pages/about.md`: homepage content
- `_pages/publications.html`: publications page
- `_publications/`: publication records
- `images/`: profile, publication, and site images
- `files/`: downloadable publication materials
